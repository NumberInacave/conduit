# conduit
fast-paced 2d momentum and movement-based platformer made with html, CSS, and JS.

### controls
w/↑ - jump<br>s/↓ - dash towards cursor<br>
a/← - move left<br>d/→ - move right<br>
space - activate grapple<br><br>
o/f - toggle fullscreen<br>p/e - toggle pause<br>
g/q - toggle grid<br><br>
click - enter fullscreen<br>
esc - exit fullscreen
